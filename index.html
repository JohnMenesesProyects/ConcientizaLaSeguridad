Los riesgos del uso de tecnologías.

Cuando navegamos por internet para realizar nuestras actividades cotidianas, como tomar una clase en linea, realizar una transacción bancaria, o cualquier actividad dentro de nuestras rutinas, se debe tener en cuenta que existen grupos de atacantes, que buscan aprovechar el desconocimiento del riesgo del uso tecnologías, para poder llevar a cabo delitos como : extorsión por robo de datos personales, daños economicos o reputacionales.   

En este post conocerás los mejores consejos para evitar caer en trampas.


¿Qué debemos hacer?

Compartimos los mejores tips para evitar el robo de datos personales, datos bancarios u algún otro daño que pueda ocasionar un uso inadecuado de internet.

Evita descargar aplicaciones de sitios no oficiales.

Los sitios apocrifos para descarga de aplicaciones puede verse como una alternativa para apps de pago, sin embargo estas pueden estar infectadas con Malware, para robo de datos, lo cual puede derivar una extorsión o perdida monetaria.

Crea una conciencia ​analítica.

No proporciones datos bancarios como nip o los 16 digitos de tu tarjeta a sitios poco conocidos, ya que podrías ser victima del robo de los mismos y sufrir una perdida economica.

Sitios inseguros

Evita navegar en páginas con el protocolo de comunicación http:

El trafico de la red en sitios con este inicio de url son vulnerables a la interceptación de los datos entre página y servidor, lo cual expone al atacante la informació que se este manejando en ese sitio.

WiFi - Maliciosos.

Evita conectarte a redes wifi gratis, ya que probablemente podrías sufrir el robo de crenciales e información confidencial.

Navega en páginas con certificados registrados

"​SSL es una tecnología estandarizada que permite cifrar el tráfico de datos entre un navegador web y un sitio web (o entre dos servidores web), protegiendo así la conexión. Esto impide que un hacker pueda ver o interceptar la información que se transmite de un punto a otro, y que puede incluir datos personales o financieros." Web Security Digicert.
